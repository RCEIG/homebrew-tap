# RCEIG Homebrew Tap

Official Homebrew tap for RCEIG tools.

## Installation

```bash
brew tap RCEIG/tap
```

## Available Formulas

### PipelineGuard
Offline-first CI/CD security scanner - catch vulnerabilities before they reach production.

```bash
brew install pguard
```

### KubeFix
AI-powered Kubernetes troubleshooting - diagnose and fix cluster issues instantly.

```bash
brew install kubefix
```

### PipelineForge
AI-powered pipeline generator - create CI/CD workflows in seconds.

```bash
brew install pforge
```

## Updates

```bash
brew update
brew upgrade pguard kubefix pforge
```
